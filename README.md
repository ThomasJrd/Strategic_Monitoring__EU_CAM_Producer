# Strategic_Monitoring__EU_CAM_Producer
Outil de veille stratégique sur la filière européenne des cathodes (CAM). Ce projet centralise les données de production 2025-2035 et analyse les acteurs clés (Orano, BASF, Umicore) via un dashboard interactif. Alliant Data Science et IA, il évalue les partenariats et risques pour anticiper les enjeux de souveraineté liés à la mobilité électrique.
